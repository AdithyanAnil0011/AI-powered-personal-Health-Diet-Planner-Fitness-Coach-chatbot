    <p align="center">
        <strong>Streamlit • Generative AI • Health Analytics • Fitness Planning</strong>
    </p>

    <hr>

    <h3>📌 Project Overview</h3>
    <p>
        This Capstone Project is an <strong>AI-powered health companion</strong> developed to address the difficulty of accessing diet recommendations, workout plans, and health insights within a single platform[cite: 2, 22, 24]. Built using <strong>Python</strong> and <strong>Streamlit</strong>, the application leverages the <strong>Google Gemini API</strong> to generate personalized routines that are practical, budget-friendly, and effective[cite: 21, 24, 32].
    </p>
    <p>
        The system integrates health metrics calculation—including <strong>BMI</strong> and <strong>daily calorie requirements</strong>—with generative AI to offer tailored 7-day meal plans and intelligent coaching based on individual user profiles and goals[cite: 31, 52, 60, 81].
    </p>

    <h3>🚀 Key Features</h3>
    <ul>
        <li><strong>Personalized Meal Planning</strong>: Generates 7-day plans with nutritional breakdowns, shopping lists, and preparation tips tailored to user restrictions[cite: 60, 61, 62, 63].</li>
        <li><strong>Food Image Analysis</strong>: Analyzes uploaded food images to provide data on calories, macronutrients, and health benefits[cite: 65, 67, 68, 69].</li>
        <li><strong>Health Metrics Calculator</strong>: Calculates BMI and categorizes users (Underweight/Normal/Overweight) to determine daily calorie needs[cite: 50, 52].</li>
        <li><strong>AI Fitness Coach</strong>: Provides personalized workout advice and motivational tips using user-specific health data[cite: 72, 74].</li>
        <li><strong>Expert Health Insights</strong>: Offers a Q&A feature for nutrition and health-related questions[cite: 28, 73].</li>
    </ul>

    <h3>🛠 Technologies Used</h3>
    <ul>
        <li><strong>Language</strong>: Python [cite: 37]</li>

        <li><strong>AI Backend</strong>: Google Gemini API (gemini-2.5-flash) [cite: 35]</li>
        <li><strong>Libraries</strong>:

    </ul>
