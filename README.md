✨ Streamlit | Personalized Health | Generative AI | Smart Meal Plans | Fitness Planning ✨  

---

## 📌 Overview

AI-Powered Personal Health & Fitness Assistant is an intelligent health companion built using Streamlit and Google Gemini API.  
It helps users get personalized meal plans, analyze food images, calculate BMI and daily calorie requirements, and receive AI-driven fitness coaching to improve overall health and wellness.

---

## 🚀 Features

* 🥗 Generate personalized 7-day meal plans with nutritional breakdowns
* 📸 Analyze food images for calories, macronutrients, and health benefits
* 🧠 Receive health insights and expert nutrition tips
* ⚖️ BMI Calculator and health status categorization
* 🔥 Daily Calorie Calculator based on user goals
* 🏃‍♂️ AI Fitness Coach with personalized workouts and motivational tips
* 🛒 Shopping list generator for meal plans
* 📊 Track health metrics and progress over time

---

## 🛠 Tech Stack

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/AdithyanAnil0011/AI-powered-personal-Health-Diet-Planner-Fitness-Coach-chatbot.git
cd AI-powered-personal-Health-Diet-Planner-Fitness-Coach-chatbot
```
---

### 2️⃣ Create a virtual environment

```
python -m venv venv
```
---

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```
streamlit run app.py
```

Your app will open in the browser automatically.
